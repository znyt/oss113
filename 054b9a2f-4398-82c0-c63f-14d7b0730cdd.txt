　　第16章
<br/>　　姓名：薛半川（小有名气的顶级谋士）
<br/>　　年龄：52
<br/>　　技能：七斩刀
<br/>　　隐藏属性：无
<br/>　　程大雷趴在草丛中，将杀人的一幕尽收眼底。对方出刀之凌厉，杀人之干脆令程大雷震惊不已，忙用系统查看对方信息。果然，这竟是一位顶级谋士。
<br/>　　程大雷知道，能达到顶级的都不简单，秦蛮这个顶级山贼能以一敌百，而一个顶级谋士显然要比一个顶级武夫更可怕。
<br/>　　薛半川忽然冷冷向这边望了一眼，程大雷一激灵，现在如果和对方碰上，自己估计只有死路一条。
<br/>　　只见薛半川忽然跃下马来，抱住韩玄之的尸体上门，手中铁刀指着来时的路。
<br/>　　“回去。”
<br/>　　令行禁止，一百匹马同时掉头，匆匆而来，匆匆又去。
<br/>　　草丛中的程大雷此时才站起来，把额头汗珠抹去：这个时代还是有高人的，也并不是每个都如徐神机那样——糟糕得算无一策。
<br/>　　忽然，刚松口气的程大雷像是响起什么，重重一跺脚，仰天发出一声长叹。
<br/>　　“我的剑呦。”
<br/>　　嘟，完成『第十一次山寨防守』，奖励抽奖机会1，是否现在抽取？
<br/>　　“先不抽，等回山寨再说。”
<br/>　　程大雷扛着自己的大斧头上马，这次系统还算够意思，肯给一次抽奖机会。抽奖是一件拼脸的事情，而程大雷的脸……最近有些黑。
<br/>　　……
<br/>　　时间：夜
<br/>　　地点：黑石城城主府
<br/>　　一张堪称可怖的大床上，有两具羊脂玉般的身子，一般身高，一般模样，一般脉脉含春的剪水长眸。
<br/>　　这便是黑石城城主最宠的一对美奴，如今二人拥在一起，像一对被剥了皮的羊羔，一具臃肿苍老的身体压在他们身上，头发花白，脸上还有点点老人斑。
<br/>　　青铜灯柱内装着灯油，将屋子照得亮若白昼，大床之上，红被翻滚，藕臂相陈，一道道挠人心窝的嗓音在软榻上响起。
<br/>　　“城主，城主……”
<br/>　　有悲呼声响起，薛半川抱着韩玄之尸体大步闯入，噗通一声跪倒在地。
<br/>　　“城主……公子，死了！”
<br/>　　“什么！”
<br/>　　韩虎踞一声惊呼，赤着身子跨过来，只见薛半川浑身浴血，腰上佩刀还滴着血，而自己的儿子脖子上出现一道伤口，早已绝了生息，临死都是一副死不瞑目的样子。
<br/>　　一张圆脸上汗珠啪啪滴落，身上肥肉随着身体颤抖而抖动，显然已经痛到极致，怒到极致。
<br/>　　薛半川低垂着头，忽然，眼中闪过一道寒芒，他握紧刀柄，斜着向上撩来。刺啦一声，在韩虎踞肚子上斩出一道伤口。他臃肿的身子踉跄倒退，靠在房间的大柱上，满脸不可思议的望着薛半川。
<br/>　　“是你？”
<br/>　　“是我！”
<br/>　　薛半川已经站起来，持刀而立，此刻他身材笔挺，表情坚硬，那还有一丝小心翼翼的模样。
<br/>　　“城主，这位置你坐得太久，已该轮到我。”
<br/>　　声音落地，刀也斩来，随着身子的旋转刀势更猛更快。韩虎踞身上一丝不挂，踉跄倒退，忽然撞到房间的青铜灯柱，他就势在地上一滚，抓起铜柱，格住这一刀，将薛半川推开。
<br/>　　“为什么，为什么！”韩虎踞以铜柱撑地，吭哧吭哧喘气。
<br/>　　“城主……乱世将至……帝国失其鹿，天下多少英雄夙兴夜寐……想着争鼎天下……而你还沉迷在酒池肉林中，这天下你不取……我取！”
<br/>　　说一声，斩一刀，七斩刀淋漓尽致斩出，屋子里只见刀光不见人影。床上那对双生子早就吓得瑟瑟发抖，谁能想到，黑石城温和儒雅的薛军师，竟然有如此武技。
<br/>　　“你想取……你也配……”
<br/>　　韩虎踞挥舞着铜柱扑过来，他身上一丝不挂，臃肿的肥肉抖动，动作竟然十分凶悍刚猛，此刻赤着身子的他，像极一只愤怒的野猪。
<br/>　　山林内一猪二虎三熊，说他是猪并不是贬义。
<br/>　　嗵！
<br/>　　铜柱砸在薛半川胸口，他踉跄后腿，口中吐出一口鲜血。
<br/>　　“不愧是三十年拒戎族于城外七日的韩虎踞，虎夫之名名不虚传。”
<br/>　　“老子用你说这些么，老子难道要靠这些才能被人害怕！”韩虎踞以睥睨的眼神盯着薛半川：“你算个什么东西，如果不是我抬举你，你就是个山中砍柴的，读了几本酸书，就想争天下，帝国的天下就算守不住，也轮不到你来争！”
<br/>　　薛半川脸上肌肉抖动，将刀柄握得更紧。而韩虎踞站在那里，依旧破口大骂。
<br/>　　“你不过就是老子养来看家的狗，连狗都做不好，就想自己做主人。我就算用脚趾头思考，也比你聪明，你就是一个自以为能用勤奋弥补愚蠢的蠢货。”
<br/>　　“城主有大才，半川自愧不如！”
<br/>　　“别他娘的酸，我告诉你，你没有那种才干，你的能力相一城都难，何况相一国，相天下，你就算累死这天下也没你份。而老子这一辈子睡过的女人比你见过的人都多，杀过的人能填满护城河，死，死又如何！而你一辈子，也只是一个累死的自虐蠢货！”
<br/>　　“杀！”薛半川眼底发赤，突然爆喝一声，持刀上前。
<br/>　　“来！”韩虎踞回以爆喝。
<br/>　　突然，一直缩在床上的那对美奴，身子如猎豹一般窜上来，一个缠住他脖子，一个抱住他大腿，束发的金钗狠狠扎入韩虎踞身体。
<br/>　　韩虎踞痛苦的发出一声惨叫，晃动身体将二女奴甩开，而这个时候，薛半川的刀已经捅进他的心脏。
<br/>　　韩虎踞的瞳孔忽然张大，身子晃了三晃，仰面倒地。
<br/>　　薛半川蹲下山，看着他的眼睛道：“我知道我不如城主，但我想试试，万一能做到呢，比如今天。”
<br/>　　“黑，黑……”
<br/>　　韩虎踞嘴巴开合，拼命想要说些什么，只是听不真切。薛半川微微皱眉，将耳朵凑在他唇边，听他说的是。
<br/>　　“黑……石城是帝国门户……不，不要让戎族……再乱……大武天下……”
<br/>　　薛半川一怔，难以理解的看着韩虎踞，只见他脑袋倾在一侧，绝了生息。
<br/>　　薛半川踏出屋外，手持虎符喝令城内军卒，那一夜，城主府被一把火化作灰烬。
<br/>　　
<br/>(本章完)
