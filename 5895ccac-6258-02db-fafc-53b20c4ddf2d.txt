　　第378章
<br/>　　奥黛琳娜，年龄19岁，身份是异教圣女，职业是顶级预言家，技能是占星，隐藏属性：无。
<br/>　　这个占星技能，每三天可以使用一次，准确率还颇高。这次来到东方，就是因为她推算到东方有诸神的归处，己方可以寻到新的机会。
<br/>　　在他们的理解中，这次长徙是神圣的、光荣的、伟大的。与奥黛琳娜同行的诸人，都是可以载入史册的光辉人物。他们将在东方传播神谕，让那些愚蠢的、麻木的、无知的人类沐浴在神光中。自己建立神殿，占有土地，享受供奉，甚至，未来将决定帝国的走向。
<br/>　　老实说，想多了。
<br/>　　所以对奥黛琳娜而言，让他们留在此地，她心里是有委屈的。不过幸好还有一点，就是程大雷允许他们自由传教。
<br/>　　奥黛琳娜向诸神祈祷，神呐，怜悯这些无知的恶人吧，就让神光先照耀他们，再照耀整个东方。
<br/>　　在这个时候，奥黛琳娜已暗暗立下誓愿，要在琴川关传教。
<br/>　　嘟，山寨有顶级神职加入，获得山寨技能：诸神的祝福。
<br/>　　程大雷正在书房翻看着这些人的资料，脑海中突然收到一条系统提示。他查看系统界面，看到山寨多了一个技能：诸神的祝福。
<br/>　　诸神的祝福是一项群体技能，CD时间是三天，每三天随机领取一项祝福。
<br/>　　比如领取到生命之神的祝福，大家的恢复能力就会增加，幸运女神是提升运气值的，大地之神的祝福可以提升作物产量……
<br/>　　有效时间是三天，三天后祝福失效，领取下一项祝福。
<br/>　　程大雷心中一动，果然就知道让这些人加入山寨是有用的，虽然没有如愿开出魔法师，但却获得了一项了不起的技能。
<br/>　　而程大雷还发现，开启攻城模式的进度条，已经提升到66%。现在程大雷已经摸索出一些经验，只要自己所做的事情是对山寨有益，进度条就会往上升，反之，进度条则会下降。
<br/>　　如此说来，距离开启攻城模式也并不遥远。
<br/>　　程大雷领取了第一次的祝福光环，获得来自爱神的祝福。
<br/>　　三天内，山寨所有人的魅力值加成，幸运值加成……这祝福光环不仅在琴川关生效，包括蛤蟆寨、落玉寨都在技能范围内，简单来说，整个蛤蟆寨都进入了发情期。
<br/>　　倒是可以趁这个机会，解决一下大家的生活问题。其实随着流民的迁入，山寨也不单是赤条条的汉子了。因为军令下达的及时，倒是没有那种欺男霸女的事情发生，不过互相之间勾肩搭背，眉来眼去的人也不少。几乎每隔几天，都有人成婚，但凡有人成亲，程大雷都有赏赐，他是格外鼓励这些事的。
<br/>　　只不过，像刘悲、秦蛮、赵子龙这些人，却还没寻得佳偶，一直都单着。
<br/>　　以他们几个的身份，在山寨还是比较受追捧的，甚至有人托媒求到程大雷身上，程大雷虽然乐意促成，可奈何神女有心，襄王无梦。
<br/>　　夜来，繁星满天，程大雷处理完山寨的事情，拐进了樊梨花屋子。
<br/>　　樊梨花已经摆好酒菜，看她脸上气色，似乎也比前些天好。大概是爱神光环发挥出来的作用，程大雷振作精神，干脆就趁着这个机会，把这件事一劳永逸的解决。
<br/>　　“今天准备好了么？”樊梨花问。
<br/>　　“当然，取酒来。”程大雷将身上的大氅挂在衣架上。
<br/>　　这几日，程大雷天天过来与樊梨花饮酒，借此开导樊梨花的情绪。一开始，程大雷还让着樊梨花，到后来，程大雷发现，不管让或者不让，似乎都没多大用。
<br/>　　樊梨花喝酒如喝水，每次醉倒的都是程大雷。晕晕乎乎的睡着，醒来后发现自己睡在樊梨花床上，樊梨花趴在桌上熟睡。
<br/>　　如此三五次，程大雷的自尊心受到很大打击，酒量也见长，可和樊梨花还有很大距离。
<br/>　　程大雷为此做了很多准备，来之前先吃饱饭，或者用浓茶解酒，但都没有多大作用。程大雷怀疑，这樊梨花是不是从小在酒坛子长大的。
<br/>　　今日他真是有准备而来，目的一定是将樊梨花喝倒，俗话说得好：酒是色媒人么，趁着这三天把事情解决了，如果过了这三天，爱神光环失效，这件事更不知何时能饮酒。
<br/>　　推杯换盏之间，程大雷自然而然就聊起奥黛琳娜的事情。这些人带来许多小玩意，看上去却也颇有趣，也拿来给樊梨花看过，只是她似乎不怎么感兴趣。
<br/>　　“听说那圣女很是漂亮？”樊梨花无意间问起。
<br/>　　隐隐约约，程大雷感觉到一阵无形的杀气。他顿了顿，漫不经心道：“是蛮漂亮，只不过没什么用。”
<br/>　　“怎么没用？”
<br/>　　“他们的信仰，圣女是不允许成婚的。”
<br/>　　总的来说，新教的教义还是很开放的，也没有什么不能喝酒，不能吃肉的规定。但对圣女的要求比较苛刻，圣女将灵魂献祭给诸神，以此获得诸神的庇佑。所以教义中规定，圣女是不可成婚的。所以不管这奥黛琳娜再漂亮，也没啥子用。真敢打她的主意，岂不是和诸神抢女人。
<br/>　　“喔，原来是这样。”
<br/>　　程大雷松口气，看来这一关自己是过去了。
<br/>　　“你为什么专门关心她能不能成亲的事情？”
<br/>　　“呃……”程大雷无言以对。
<br/>　　时间一分一秒过去，房间内的油灯也续了三次。樊梨花觉得有些奇怪，按程大雷以前的酒量，此刻早该醉倒，怎么今日的他，不仅没有醉，反而看上去越来越精神。
<br/>　　程大雷是有准备而来，奥黛琳娜带来一种转心壶，壶中放两种酒，一种是度数高的烧酒，一种是度数低的米酒。程大雷都是喝得米酒，几斤是没问题的，烧酒就更烈一些。
<br/>　　樊梨花昏昏沉沉，意识已有些不清楚，看程大雷已经出现几分重影。
<br/>　　程大雷鬼鬼祟祟，揽住樊梨花的肩膀：“你醉了，我扶你上床休息。”
<br/>　　“我……我……”樊梨花神识不定，看程大雷的眼神又爱又恨，忽地一发力，结结实实一个过肩摔将程大雷摔在地上。
