<br/>    至于说什么自己真的很喜欢修行，别扯淡了，不存在的。
<br/>    
<br/>        陈明也是开始思考自己如今的最大的问题，如今的霸仙剑，自己只是修到了一个及格分，那说明霸仙剑还能更强，自己目前做到的就是将两剑融合起来，完美融合到了一处。
<br/>    
<br/>        但是不论怎么说，这依旧是两剑，而不是一剑。
<br/>    
<br/>        陈明瞬间明白了过来，自己是要把两剑修成一剑，这才是真正的霸仙剑。
<br/>    
<br/>        如今陈明已经能够将两剑完美融合，现在需要做的，便是让这两剑水乳交融。
<br/>    
<br/>        说实话，如果不是早就知道这么干真的可以，陈明打死也不信霸道剑和仙道剑可以融合为一剑。
<br/>    
<br/>        苦修的剑法都是不脱手的剑法，法修的剑法，一般都是飞剑，想要融合，确实很难。
<br/>    
<br/>        不过既然已经有人走通了这条路，而且走通的人还不少，这就说明是可行的。
<br/>    
<br/>        而枯荣道最大的作用就是告诉自己，自己修行的方向没有错。
<br/>    
<br/>        若是错了，枯荣道自然会给自己一个低分，若是对了，那就会分数变高。
<br/>    
<br/>        时间在缓缓的过去，陈明也是开始认真地融合这两剑为一剑，而现在的陈明，也是开始思考更深层次的问题，他现在施展，依旧是用法力施展一剑开天，气血之力施展周身八尺无敌，这肯定是不对的。
<br/>    
<br/>        自己必须要做到气血与法力交融。
<br/>    
<br/>        索性还好的是，陈明的道宫很多。
<br/>    
<br/>        遍布周身，所以这一环对于陈明来说并不是很难。
<br/>    
<br/>        而陈明接下来要做的，便是心中想着这只是一剑，而不是两剑，真正融合为一剑。
<br/>    
<br/>        时间就这么缓缓地过去，陈明继续在枯荣道中修行霸仙剑。
<br/>    
<br/>        但是星空，可从来不会平静。
<br/>    
<br/>        零号站在自己的大殿之中，在他的身前，无免者一个接着一个出现，这是他的属下，也是他成为零号无眠之后的第一批无免者。
<br/>    
<br/>        零号先是看了众人一眼，他带着睁着一眼，闭着一眼的面具，这个面具
<br/>    
<br/>        的意思是总有一只眼睛看着这世间，所有的一切都无法逃过他的眼。
<br/>    
<br/>        零号道：“首先，我要说一点，我的麾下，不能出现五号无面者，明白吗？”
<br/>    
<br/>        众多无免者都是齐齐拱手道：“我等遵命！”
<br/>    
<br/>        零号继续道：“神山，这个时代，居然还有着人想要窃取天道的权柄，若不是我修为太高，也想去较量一番了。”
<br/>    
<br/>        有人道：“无眠者阁下，我们已经派人去了！无冕者议会四大首席，以及首席之上的武夫，也已经派出去了。”
<br/>    
<br/>        零号道：“看来，你们并不明白这无字劫到底意味着什么，若是无法突破极限，根本没有半点用处。武夫那个小子，确实实力不错，不过，可能还是不够保险。”
<br/>    
<br/>        有无免者道：“这世上，地仙之中，莫非还有着比武夫更厉害的地仙？”
<br/>    
<br/>        零号笑了笑，道：“当然有了，我就是那个地方的弃徒，你说有没有？”
<br/>    
<br/>        众多无免者都不敢说话，赫赫威名，即便是四大仙国，都不敢动的零号无眠，居然是某个地方的弃徒！
<br/>    
<br/>        这实在是太让人惊讶了。
<br/>    
<br/>        零号继续道：“在一颗偏僻的星辰之上，有着一个名为天道院的道观，我需要一个人，去送一个口信过去。”
<br/>    
<br/>        有无免者道：“无眠者阁下，我们不是怕死，只是，这样的地方，我们真的进得去吗？”
<br/>    
<br/>        零号道：“进去？你们自然不可能进得去。所以我只是要你们其中一个人带个口信。我记得，小师弟的修为，似乎是差不多可以和这个渡劫者一战了，去吧。”
<br/>    
<br/>        数月之后。
<br/>    
<br/>        天道院外，一个无免者带着愤怒的面具站在天道院外，他从来没有想到过，天道院，居然就是这么一颗弱小之极的星辰，这颗星辰之上，没有任何修行者，以至于，神山都没有此处的投影。
<br/>    
<br/>        此处，看不到神山。
<br/>    
<br/>        天道院，在一个偏僻之极的小渔村外，无免者没有敢进去，只是站在天道院外，静静地等候，太阳出来了，温暖的阳光洒在大地上。
<br/>    
<br/>        从中，走出一个年轻的道士，拿着扫帚开始扫地。
<br/>    
<br/>        无免者看向这个年轻的道士，完全看不出对方的修为，对方，如同就是一个凡人。
<br/>    
<br/>        无免者看向这年轻的道士，静静道：“无字劫降临了。”
<br/>    
<br/>        年轻的道士依旧是扫地，没有任何反应，无免者还以为这年轻的道士没有听见，继续道：“无字劫降临了。”
<br/>    
<br/>        年轻的道士看了他一眼，让他觉得浑身发寒，但是实际上，这个年轻的道士看向他的目光很温和，温暖如同春风。
<br/>    
<br/>        “我知道了。”
<br/>    
<br/>        无免者想要尽快离开这个地方，他并不想在此地久留，无眠者，不会拿自己的出身开玩笑。
<br/>    
<br/>        年轻的道士继续道：“是大师兄让你来的吧？”
<br/>    
<br/>        无免者不知道说什么，他并不知道零号在这天道院的排行，灰溜溜地走了。
<br/>    
<br/>        年轻的道士看向周围，继续扫着他的地，待得他已经将所有的落叶都扫走。
<br/>    
<br/>        他下山，来到一个小渔村之中，这里，嬉笑打闹的孩子有不少，但是见到他之后，都是学着书生的样子拱手道：“子鱼先生。”
<br/>

<br/>    
<br/>        子鱼站在沙盘前，将这些字都写在沙盘上，不过一会儿，院子里传出了郎朗的读书声。

<br/>        三个时辰之后，子鱼看向这些孩子，已经是心不在焉，这样的小孩，心思不集中乃是正常的，继续教下去，也没有什么用了。

<br/>        “都出去玩吧。”
<br/>    
<br/>        子鱼离开了院子，走向旁边，在旁边的院子里，住着一个寡居的女人，叫做杏花。子鱼站在女人的院子前，轻轻敲了敲门。
<br/>    
<br/>        杏花有些憧憬地看向院门，伸手打开了院门，“子鱼先生！”
<br/>    
<br/>        言语中有着惊喜，同样有着一丝羞涩。
<br/>    
<br/>        子鱼微笑道：“我算过的，你的水缸应该已经见底了，我索性也是无事，帮你挑满水缸吧。”
