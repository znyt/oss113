　　等待了半个多小时的功夫，道路的尽头驶来一辆黑色的轿车，似乎司机十分赶时间一般，车速极快。
<br/>　　这样偏僻的地方，很少有车辆和行人到来，虽说是路，但也早已失修，像这样的高级轿车，几乎是不会出现在这里的。
<br/>　　如果没有特殊的事情，没有哪位车主舍得将车开到这种地方，万一剐蹭了一下底盘，也很让人心疼。
<br/>　　从藏身的地点走出，刘哔已经收起了翅膀，他现在的外形，和人类没有太大差别。
<br/>　　“嗤！”
<br/>　　高速刹车，让轿车往前惯性的向前滑行了十几米，轮胎和地面摩擦，带起一片飞扬的尘土，空气中，弥漫起了一股刺鼻的焦糊味。
<br/>　　轿车停在刘哔面前，黑色的车门打开，下来一个穿着考究的紫发男子。
<br/>　　紫发男子满脸慌张，下车后并没搭理刘哔，而是四处打量了一下，目光停留在还在昏迷状态的金木研身上。
<br/>　　“哦！我亲爱的金木，我只不过是离开了你几天，为什么，你就被搞成了这样！”脸上带着悲痛欲绝的表情，男子快速来到金木身边，仔细检查着他身上的伤势。
<br/>　　看着蹲在自己面前的紫发男子，刘哔嘴角露出一抹笑意，‘美食家’月山习，S级喰种，也是月山家的唯一继承人。
<br/>　　“喂，月山习，提醒你一下，我们刚从库克利亚逃出，如果不抓紧时间离开这里的话，CCG会追上来的。”走上前去，刘哔轻轻拍了拍月山的肩膀，开口说道。
<br/>　　月山习身体僵了一下，转过头来，目光有些深沉，脸上却再也没有开始那般浮夸的表情了。
<br/>　　两人对视了几秒，月山似乎想说什么，最终却闭上了嘴巴，抱起金木研回到了车里。
<br/>　　他知道现在不是说这些的时候，在CCG到临之前，他需要一些时间来消除痕迹，否则话，一旦事情败露，家族可就十分危险了。
<br/>　　刘哔也没有继续说话，抱着什造上了车，耐心等待起来，他知道月山会把一切事情都解决好的。
<br/>　　片刻，本来下车清扫痕迹的司机回到了车里，打开车载电脑点选了几个按钮，转过头来，示意一切已经没什么问题了。
<br/>　　月山点了点头，正准备让司机撤离，鼻尖却闻到一股烟草的味道。
<br/>　　“喂，在别人车里抽烟，可不是一个好习惯。”轻轻皱了皱眉头，月山习十分讨厌这种呛人的味道。
<br/>　　手里夹着刚刚点燃的香烟，刘哔动作一顿，无奈的耸了耸肩。
<br/>　　“不好意思，那我掐了好了。”深深的吸了一口，刘哔摇下玻璃，向外吐出一股烟雾，就要将香烟给弹飞出去。
<br/>　　前方的驾驶位上，司机向后伸手，阻止了刘哔的动作，随后递过来一个玻璃制的烟缸，示意他将香烟掐灭在这里面。
<br/>　　他费尽心思才清理好现场，自然不可能留下任何可能暴露出去的痕迹。
<br/>　　不满的嘀咕一声，刘哔也没有反对，灭掉手上的香烟后，摇上了玻璃。
<br/>　　轿车缓缓发动，顺着来路驶去，片刻之后，彻底驶离了这处危险的地方。
<br/>　　司机和月山习都没发现的是，在刘哔之前打开车窗时，一张名片自他袖口处掉下，飘落在了坑坑洼洼的地面上。
<br/>　　这张名片，是月山在古董送给金木研的那张，后来因为什造偷走了金木的钱包，这张名片自然也就落入了什造手中。
<br/>　　而对它没什么兴趣的什造，将名片丢弃之后，被刘哔恰好看到，收了起来，一直存留到现在。
<br/>　　当时他可没想那么多，他收集这张名片，只是为了得到月山的联系方式然后找机会将他吞噬。
<br/>　　但吞噬芳村功善的尸体之前，他并没有击杀S级喰种的实力，所以才一直没有寻找月山，不曾想的是，之后却派上了用场。
<br/>　　相比于吞噬，他有更好的方法，将月山的资源给利用起来。
<br/>　　黑色轿车一路平稳行驶，透过外封闭的玻璃，刘哔注意到汽车已经驶入了市区，街道上也开始热闹了起来。
<br/>　　看着熙熙攘攘的人群，他突然有种梦幻的感觉，天知道他在那小黑屋里带了多长时间，他现在的感觉，宛如做梦一般。
<br/>　　虽然让CCG将自己抓进去是早就有的计划，但所受的折磨却是实打实的，要不是他有系统可以玩，估计现在都快要疯掉了。
<br/>　　就算是修炼，也根本阻挡不了那份孤寂，那份，万物尽灭般的死寂。
<br/>　　大概一个多小时后，轿车驶入了一处偌大庄园，这个地方他没来过，但从周遭成群的别墅看来，能居住到这里的身份应该都不简单。
<br/>　　说是宅子，但里面的面积几乎快能敢上末世前的中学一般大小了，汽车减缓速度，行驶在偌大的院子里，路边的花圃中，开满了五颜六色的鲜花。
<br/>　　最终，车子停在一栋仿古的三层城堡前，刘哔抱起什造，跟随月山一起下了车，仔细打量了一下眼前的城堡。
<br/>　　眼前的城堡不太像是日系风格的建筑，反倒有种复古的英伦风，其墙上脱落的表皮，更显露出了一丝沧桑的历史感。
<br/>　　虽然总体来看整个城堡没什么问题，但仔细研究就能轻易察觉到，那些经过修补的墙体，这个地方，不知道存在了多长时间。
<br/>　　“习大人，您回来了。”一个看起来并不大的男孩开了门，梳着中分，发色和月山习一样同为紫色。
<br/>　　点了点头，月山习没有多说什么，怀中抱着金木，急匆匆的走了进去。
<br/>　　他现在十分担忧金木的安危，对现在的他来说，金木研不仅仅是食物那般简单了。
<br/>　　这种感觉，更像是一股执着，或者说近乎于变态的留恋，当看到金木受伤的时候，他的内心就根本无法平静下去了。
<br/>　　美食固然重要，但终究，还有比美食更加重要的东西。
<br/>　　该死，究竟是谁将金木弄成了这样，要是让我查出来，一定不会轻易放过你！
<br/>　　月山习脸色阴沉，之前给他开门的紫发男孩跟在众人身边，低着头不敢言语，他也察觉到气氛有些不太对劲。
