　　天海市，江北区警察局，审讯室，
<br/>　　江飞燕今天很郁闷，
<br/>　　半个小时前，她接到报案，有人行凶伤人，于是便立即带手下缉拿凶手。
<br/>　　一切都很顺利，他们只用了十分钟的时间，就将嫌疑人捉拿归案。
<br/>　　而且，三名嫌疑人也都十分配合，乖乖跟着他们回到了警局。
<br/>　　眼看着她的破案纪录，就要再次被刷新，让江飞燕没有想到的是，这个嫌疑犯却软硬不吃的，任凭她使出了所有审讯手段，对方就是不开口，而且态度极为淡定，甚至可以说是嚣张，完全不把他们放在眼里！
<br/>　　这不，足足审讯了半个小时，却愣是没有一点进展。
<br/>　　“小子！你别以为你不说话，我们就定不了你的罪！你的另外两个同伴，可已经全都招了！”
<br/>　　“就是你动手打伤了朱博文，还有雅尚私人会所十几名工作人员！”
<br/>　　“我查过你的资料，你今年刚满十八岁，已经具备了民事责任能力和刑事责任能力，就凭你故意伤人致人重伤这一条，就足以让你蹲大牢！”
<br/>　　“只要你主动坦白，就可以考虑减刑，一味的不配合，只会加重你的刑罚……”
<br/>　　江飞燕说的口干舌燥，叶尘却依旧一脸悠然，翘着二郎腿，浑然不觉得自己已经大祸临头。
<br/>　　和江飞燕一同审讯的青年男子警察，顿时气得肺都要炸了，直接一拍桌子，
<br/>　　“小子！你特么再不说话，信不信老子这就把你直接关进拘留所？”
<br/>　　也不知道是不是青年警察的这话起了作用，叶尘终于缓缓抬起头来，不过等他一开口，两人顿时听懵了，
<br/>　　“两位警官，不如我们来打个赌吧！10秒之后，你们局长会走进来，亲自帮我解开镣铐，并且恭恭敬敬把我送出警局！”
<br/>　　江飞燕和那名青年警察全都一愣，继而那名青年警察，已经气极反笑，
<br/>　　“小子！你是得了妄想症吗？怎么？想冒充精神病来逃避刑罚？我告诉你，门都没有！”
<br/>　　青年警察这话才刚刚落下，
<br/>　　嘎吱~
<br/>　　审讯室的门被人推开，一名四十来岁、透着威严之气的中年男子，身穿高级警官的警服，从外面快步走了进来，
<br/>　　江飞燕两人看到此人出现，顿时全都惊呆了，不自觉的就站了起来，恭恭敬敬喊了一声，
<br/>　　“张局！您……您怎么来了？”
<br/>　　然而他们的顶头上司张局，只是冲他们随意摆了摆手，已经满脸赔笑的朝叶尘走了过去，直接鞠躬行礼，
<br/>　　“叶先生！实在抱歉！手下人不知道您的身份，冲撞了您，您可千万不要放在心上！”
<br/>　　张局一边说着，回头冲着已经完全傻掉的江飞燕两人，脸色一板，
<br/>　　“你们都还愣着干什么？还不赶紧把钥匙拿出来，给叶先生解开手铐！”
<br/>　　江飞燕和那名青年警察，此时大脑完全处于短路状态，已经彻底懵逼了。
<br/>　　事情的发展，竟然和叶尘之前说的一模一样！
<br/>　　‘难道这小子，是某位权贵公子哥？可是这也说不通呀！’
<br/>　　要知道，他们的这位的局长，向来以嫉恶如仇而著称。
<br/>　　记得前几年，一位省级大员的公子，在江北区犯了事，他们这位张局直接亲自动手抓人，一点都不含糊。
<br/>　　这样一位刚正不阿的警察局长，居然问都不问，就要把眼前的嫌疑人释放？
<br/>　　他们实在想不通！
<br/>　　“张局，这小子把人打成重伤，怎么能……”
<br/>　　江飞燕还想理论，不料却被张局一口打断，
<br/>　　“什么这小子，那小子，这位是叶帅！是我们华夏的顶梁柱……”
<br/>　　还没等张局说完，叶尘又忽然开口打断，道：
<br/>　　“算了！张局是吧？他们也是依法办事，不知者不罪，你也不必追究了！”
<br/>　　叶尘一边说着，两手轻轻一挣，
<br/>　　彭！
<br/>　　拷在叶尘手腕上的手铐，直接应声而断。
<br/>　　江飞燕两人，再次满脸骇然，显然没有想到，眼前这如此年轻的少年，竟然有着如此恐怖的实力！
<br/>　　叶尘伸了一个懒腰，
<br/>　　“张局，我现在可以走吗？”
<br/>　　张局连忙侧身站到一旁，
<br/>　　“当然！叶帅请！”
<br/>　　直到这时，江飞燕两人这才听清楚，张局对眼前这个嫌疑人的称呼。
<br/>　　叶帅！
<br/>　　两人顿时更加惊疑不定，
<br/>　　叶帅？
<br/>　　他的名字不是叫叶尘吗？难道是他的绰号？
<br/>　　也难怪两人想不通，
<br/>　　华夏自从建国时，授予过十位战功卓越的元帅，之后的数十年，别说元帅了，便是大将也没有再出现过！
<br/>　　和平年代，军中最高军衔也就是上将，哪里来的元帅？
<br/>　　在两人的震惊和疑惑中，叶尘已经缓步走出了审讯室，张局则恭恭敬敬的跟在身后。
<br/>　　江飞燕两人，虽然心中依旧大为不解，不过已经不敢阻拦，当即也快步跟了上去。
<br/>　　“张局，我的那两位朋友……”
<br/>　　出了审讯室之后，叶尘转而又冲张局望去，后者当即道：
<br/>　　“叶帅放心！我这就派人去请！”
<br/>　　叶尘点了点头，不再多说。
<br/>　　而就在这时，
<br/>　　忽然一道粗狂的声音，在整个警局陡然响起，
<br/>　　“我倒要看看，是哪个不长眼的，连叶帅都敢抓？”
<br/>　　声音如雷，整个警局的人，都感觉脑袋嗡嗡作响，耳膜生疼。
<br/>　　继而，一道身材高大笔直，一身军装的中年男子，从外面走了进来，
<br/>　　“两杠三星！居然是一名上校！”
<br/>　　“他刚刚那话是什么意思？叶帅是谁？”
<br/>　　“这些丘八可不好惹，而且此人军衔这么高，我们警局怕是要有麻烦了！”
<br/>　　……
<br/>　　在众人的议论声中，只见那名中年上校，已经大踏步走了进来，然后在人群中一番扫视，两眼忽然一亮。
<br/>　　继而，在众人充满疑惑的目光中，朝一名身穿休闲服的少年，快步走了过去。
<br/>　　所有人正疑惑不解，这位上校接下来的举动，顿时让所有人都看懵了！
<br/>　　只见，这位上校快步走到那名少年跟前，直接立正，行了一个标准的军礼，
<br/>　　“叶帅好！”
<br/>　　这一刻，整个警局，一片寂静。
<br/>　　之前审讯叶尘的江飞燕两人，更是两眼瞪得老大，简直都有已经开始有些怀疑人生了。
<br/>　　又是叶帅！
<br/>　　难道他真是……这怎么可能？
