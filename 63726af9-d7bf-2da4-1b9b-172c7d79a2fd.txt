　　好在这吞天蟒虽然是五阶妖兽，却并不以速度见长，虽然九翅紫电鹫载着两个人，在速度还是稳占上风，但想要将其甩开，一时半会也难以做到。
<br/>　　一转眼，大半天时间过去了，那吞天蟒也不知道为何，竟然一直紧追不舍，而九翅紫电鹫则开始有些体力不济，尽管在拼命的闪动着翅膀，两者的距离却依旧在逐渐拉进。
<br/>　　“不行！我撑住了！”
<br/>　　九翅紫电鹫终于支撑不住，开始朝下方坠落。
<br/>　　叶尘见状，不由得摇了摇头，
<br/>　　他也没有想到，这吞天蟒竟然追了足足十几万里还不肯罢休。
<br/>　　“你们在前面等着我，我去会会它！”
<br/>　　说着，叶尘直接纵身一跃，自九翅紫电鹫的背上跳下，
<br/>　　“紫琼剑！”
<br/>　　轰隆！
<br/>　　一道长达超过千米的紫色剑芒，突然冲天而起，狠狠斩落在后方的那头吞天蟒的身上。
<br/>　　“嗷！！”
<br/>　　那头吞天蟒显然也没有想到，前面的人居然会转头攻击它，顿时负痛嚎叫了一声，连同那庞大的身躯，也被叶尘这一剑砸落到了地面之上。
<br/>　　轰隆隆！
<br/>　　那长达上百米的庞大身躯，狠狠砸落在地上，下方顿时犹如发生了一场大型地震，而且出现了一个方圆数千米的深坑！
<br/>　　然而，叶尘却眉头微微一皱，脸上露出一抹凝重之色，
<br/>　　要知道，他刚刚这一剑，几乎已经动用了他八九成的功力，而且又是突然袭击，若是换成九翅紫电鹫，此刻早就被他一剑劈成两半了。
<br/>　　可是这头吞天蟒，却只是身上出现了一道血痕，甚至连轻伤都算不上！
<br/>　　‘这畜生的防御太高，看来普通的物理攻击，对它没有太大效果！’
<br/>　　而那头吞天蟒，坠落地面之后，发出一道吼叫声，很快便冲天而起，愤怒的咆哮着，朝叶尘猛扑了过来！
<br/>　　“琉璃金身！”
<br/>　　“法相天地！”
<br/>　　叶尘没有时间去思索，果断同时使出两门武技。
<br/>　　轰！
<br/>　　那吞天蟒冲了空中，身体陡然一扭，那硕大无比的尾巴，犹如一根长鞭，狠狠朝叶尘砸落下来，速度快到不可思议！
<br/>　　“哼！”
<br/>　　叶尘冷哼一声，当即一拳迎了上去。
<br/>　　轰隆！
<br/>　　天空一道爆炸声响起，叶尘的拳头，和吞天蟒的尾巴在空中撞击在一处。
<br/>　　轰！
<br/>　　叶尘瞬间便倒飞了出去，足足退到千米之外，方才站稳身形。
<br/>　　很显然，若是拼力量的话，身为人类之躯的叶尘，远远不是眼前这个五阶妖兽的对手。
<br/>　　纵然他吸收了神龙之血，又精通炼体之术，力量远远超过一般的修真者，可是和气血雄厚的吞天蟒相比，终究有着不小的差距。
<br/>　　“咳咳！”
<br/>　　叶尘不由得咳嗽两声，体内一阵气血翻滚，险些喷出一口鲜血出来。
<br/>　　而此时，已经逃到数十里开外的九翅紫电鹫，已经幻化成了一个紫衣老者的模样，和一旁的北山薇儿，两人全都脸色大变。
<br/>　　北山薇儿忽然心念一动，扯了扯紫衣老者的衣袖，道：
<br/>　　“九爷爷，趁这小子和吞天蟒正在打架，要不我们赶紧趁机逃回北寒城吧？”
<br/>　　不料，紫衣老者却摇了摇头道：
<br/>　　“不可！薇儿，此子的实力深不可测！我们必须要先确定，这小子死在吞天蟒的手中，否则他若是杀回北寒城，我们将有灭城之灾啊！”
<br/>　　北山薇儿听到这话，又想起之前叶尘的种种手段，顿时一阵颓然。
<br/>　　“不过……”
<br/>　　九翅紫电鹫接着又道：
<br/>　　“这吞天蟒乃是五阶妖兽，是这片冰原森林当之无愧的霸主，便是化神天君都不敢轻易招惹，这小子虽然离开，却也未必能够全身而退！”
<br/>　　北山薇儿听到这话，美眸顿时微微一亮，心中又燃起了希望。
<br/>　　而与此同时，
<br/>　　数百里之外，有着一艘华丽庞大的金色飞船，足足有数千米长，数百米宽，而且周身玄光流转，隐隐还包含各种阵法禁制。
<br/>　　很显然，这是一件飞行类的法宝，而且等级还颇为不低。
<br/>　　而此时，飞船之上，一群人正站在凭栏处，朝叶尘和吞天蟒交战的地方观望。
<br/>　　“这气息，很显然是那头冰原森林的霸主，吞天蟒！”
<br/>　　“和那吞天蟒交战的，似乎是一个人类修士？”
<br/>　　“放眼整个天玄星，够资格和这头吞天蟒一战的，也不过只有十几人而已，而且每一位，都是我们天玄星赫赫有名的大人物！”
<br/>　　……
<br/>　　众人议论一阵，站在众人最前方的一名青年男子，一脸兴奋道：
<br/>　　“此等大战，百年难遇！既然我们遇上，怎么能错过？留下一部分一人，看守追风穿云舟，其他人随我前去看看！”
<br/>　　那青年男子说着，直接纵身一跃，便从飞船上跃出，朝叶尘和吞天蟒交战的方向飞掠而去。
<br/>　　其余众高手见状，连忙纷纷跟上。
<br/>　　不一会，
<br/>　　众人来到数十里外的一座山峰，对战场中的情况，自然也就看到越发清楚。
<br/>　　“我没有看错吧？和吞天蟒打的难分难解的那个人，竟然是一个如此年轻的少年？”
<br/>　　“难道此人是某位精通驻颜之术的老家伙？我天玄星上，可没听说过，有这号人物！”
<br/>　　“或许是来自外星域，又或者是某位隐世高人？”
<br/>　　……
<br/>　　众人议论纷纷，开始纷纷猜测叶尘的身份来历。
<br/>　　“雷叔，您怎么看？”
<br/>　　那名青年男子，转而望向身旁，一名不怎么说话的老者，忽然开口问道。
<br/>　　其余众人，似乎对这老者，也颇为敬畏，于是纷纷闭嘴，朝他望了过来。
<br/>　　那位被称作‘雷老’的老者，并没有着急开口，而是两眼死死盯着叶尘片刻，忽然脸色露出一抹震惊之色，
<br/>　　“咦！奇怪！真是奇怪！”
<br/>　　那青年男子道：
<br/>　　“雷叔，您是不是发现了什么？”
<br/>　　老者又深吸了一口气，方才道：
<br/>　　“你们仔细看，虽然此子的气息雄浑，实力也强的可怕，竟然可以和五阶妖兽的吞天蟒对抗，可是他所运转的真元，仅仅只是金丹之力而已！”
<br/>　　“什么？这怎么可能！”
<br/>　　所有人听到这话，全都两眼一瞪，
<br/>　　修真者达到不同的境界，所使用的真元之力，是有着细微的差别的，只不过很少有人会关注这一点。
<br/>　　如果一个人所使用的真元，包含的金丹之力，那也就意味着，他只有金丹境的修为，这是很难伪装的。
<br/>　　以金丹境的修为，对抗五阶妖兽的吞天蟒，这有可能吗？
<br/>　　众人顿时相顾骇然。
