　　第504章
<br/>　　程大雷赤膊扛着聂隐娘，深一脚浅一脚走出大山。此刻夜已经黑得深沉，今天没有月亮，繁星满天。
<br/>　　程大雷深吸一口气，吐出肺腑中的浊气。这要是一路走到山寨，还不知到什么时候。说也奇怪，被聂隐娘一路追杀出上百里，程大雷未必觉得累，但若让他徒步走回山寨，程大雷却觉得疲惫不堪。
<br/>　　他心中一动，在系统商店兑换了一匹马。等不多时，一匹战马从夜幕中奔过来，走到程大雷面前，打了个响鼻，恭顺的将头垂下。
<br/>　　这一幕当真是将聂隐娘骇得不轻，这匹马自然不会是程大雷事先留在此地，但看他脸上的表情却一点也不觉得奇怪。外面都传说程大雷有什么鬼神不测的手段，现在看来，传言也未必是空穴来风。
<br/>　　程大雷将聂隐娘搁在马背上，动作自然谈不上有多温柔，聂隐娘浑身骨骼酸疼。她若有半分力气，甘愿自戳解决性命，但现在她受制于人，连自尽的力气都无。
<br/>　　程大雷骑马押着聂隐娘返回山寨，大概走出半个时辰，就看到前面火光飘忽。有一队人骑着战马，打着火把向这边奔来。
<br/>　　程大雷口中一声呼哨，将人引到这边来。为首之人喊道：“是大当家么，是大当家么？”
<br/>　　“是秦蛮么？”
<br/>　　穿透夜幕，秦蛮带着一队人迫近，待看到程大雷后，上三眼下三眼的打量他一遍，口中道：“啊呀呀，大当家没遭了那娘们毒手吧？”
<br/>　　“什么话！”程大雷指了指马上的聂隐娘：“本当家还能被一个女人害了，无非是顾忌她是个女流之辈。喏，这不被本大家轻而易举的擒了。”
<br/>　　秦蛮撇撇嘴，心道：原不是你被吓得当缩头乌龟的时候。当然，这时候他不会对程大雷说这些话，口中道：“回来便好，回来便好，山寨现在都在寻大当家，既然大当家平安回来，我们便一起回城。”
<br/>　　程大雷挥挥手：“走着。”
<br/>　　还有一头大虫一条怪蛇的尸体在大山之中，程大雷自觉丢了可惜，差遣小喽啰去山中取来。他自己和秦蛮一行人返回蛤蟆城。
<br/>　　程大雷离去的这大半天时间，山寨里的诸人发了疯，能动弹的都动了，派出一队队人在草原上寻找。不如此，程大雷也不可能这么快就碰上秦蛮。
<br/>　　现在既然寻到了程大雷，虽然他现在有些狼狈，但终究没受什么伤。秦蛮便差人传讯，把撒在草原上的人手召回来。
<br/>　　直到次日天明，程大雷才算是返回蛤蟆城。徐神机、刘悲、和珅等人见到程大雷回来，自然是人人开心，个个欢喜。
<br/>　　“来人。”程大雷唤了一声：“先将这姑娘关起来。”
<br/>　　对于聂隐娘的本事，程大雷一直都不敢小看。他虽然已疲惫不堪，但仍拖着疲惫的身体，亲自派人将聂隐娘关起来。
<br/>　　将聂隐娘押入大牢，身上上了手铐脚镣，又用大枷绑着，免得这姑娘越狱。
<br/>　　之后，程大雷才回到城主府，先洗掉身上的血污，在热水中好好泡了一阵，换过干净衣服。之后又喝了一些热粥，去掉连日以来的燥气，这时骨子里的疲惫感才涌上来。程大雷连眼睛都睁不开，一头扎在床上睡死过去。
<br/>　　毕竟是睡卧马鞍桥，渴饮刀头血的人物，这一夜奔波虽然累，倒是也折磨不垮程大雷，主要是连日以来的精神压力太过熬人。如今好不容易擒住聂隐娘，程大雷才能把肩上的压力卸下去。
<br/>　　这一觉睡得好沉，一直从黎明睡到月上中天。程大雷晕晕乎乎醒来，都感觉脑袋冲血，天南地北分布清楚。
<br/>　　苏樱端着醒神茶进来，程大雷抿了一口，才算是恢复些精气神。算起来，也是有月余没见过苏樱，连日奔波，这日子当真不是人过的。
<br/>　　不过好在是生擒了聂隐娘，看那姑娘的意思，也是脑袋有些不清楚。只要自己巧用三寸不烂，口若悬河巴拉巴拉忽悠一通，想把那聂隐娘收付不难。
<br/>　　若能是将聂隐娘收付，天下何人不能杀。纵然不派她出去杀人，贴身保护总是不差的。程大雷心里也明白，诸王分立，日后暗杀行刺这种事情肯定少不了。有聂隐娘在身边保护，程大雷不相信谁可能杀了自己。
<br/>　　程大雷捧着茶，心中正想得得意，时不时还发出几声瘆人的冷笑。苏樱不知发生了什么事，问道：“你怎么了？”
<br/>　　“没什么啊，我很好，没……”
<br/>　　一瞬间，程大雷突然脸色发白，脸色变得极其阴沉难看。
<br/>　　就在刚刚，程大雷收到一条系统提示。
<br/>　　嘟，恭喜击杀聂隐娘，获得抽奖次数1
<br/>　　程大雷心中一激灵，忙从床上站起来，起身就往外走。苏樱不知发生了怎样事，拿了件袍子跟在程大雷外面。
<br/>　　程大雷是一路走到大佬，大佬门外两个卫兵见到程大雷忙挺直了腰杆。
<br/>　　“大当家。”
<br/>　　“把门打开，里面关着的人没事吧？”
<br/>　　“没事啊，我们兄弟晚上还进去送了饭，怎么会有事。”
<br/>　　地牢打开，程大雷和苏樱一路走到聂隐娘的牢房前，只是看了一眼，苏樱便闭上眼睛，悄悄将头侧到一边。
<br/>　　只见聂隐娘口中流出黑血，脑袋耸拉下来，很明显已经气息断绝，一命呜呼。
<br/>　　两名喽啰大吃一惊，结结巴巴道：“大当家，她怎么死了，我们刚刚还……”
<br/>　　因为之前已收到系统提示，所以程大雷也并不觉得奇怪，只不过过来看一眼确认而已。
<br/>　　不过他也有些疑惑，这姑娘究竟怎么死的。
<br/>　　让蛤蟆城的仵作看过，才明白是什么原因。如这样的杀手，身上都带着毒药，一是为了免得被抓后泄露情报，二也免得遭受严刑拷打。
<br/>　　毕竟，这世上让一个人生不如死的法子太多了，尤其是一个女人。
<br/>　　这些毒药往往藏在指甲缝、牙齿、头发中，而如聂隐娘身上的毒药，是藏在一头黑发中。被抓进地牢中后，她思前想后，觉得与其受辱不如慷慨一死，所以服毒自尽，一命奔了黄泉。
<br/>(本章完)
