　　翌日，清晨的和煦阳光透过窗户照射入房。
<br/>　　那充满了生机的光芒照耀在桌旁慵懒少年的俊俏脸庞上，给人一种很是梦幻的感觉。
<br/>　　但谁又知道，这慵懒的外表下，隐藏着一颗多么坚强的心；这翩翩公子一般的俊朗飘逸面容下，隐藏着一个桀骜不羁的灵魂。
<br/>　　“嗯？”萧逸感受到自己身上暖暖的阳光，微微醒了过来。
<br/>　　昨晚，依依睡床，他则随便趴在桌面上休息。
<br/>　　依依自己也有一间偏房，但那是下人的房间，远不如萧逸的房间舒服。她有伤在身，萧逸自然不会让她睡回偏房。
<br/>　　当萧逸醒来，已经发现，桌面上摆满了早餐。
<br/>　　食物并没有多么丰盛，却很诱人。
<br/>　　一碗白粥，但浓稠得当，米粒与水完美地融合了在一起，粥面飘着淡淡的清香，必然香滑可口。
<br/>　　几个馒头，但被蒸得松软适中，洁白的表面散发着热腾腾的热气，让人食欲大增。
<br/>　　“少爷，你醒了。”依依恰好走进来，手上还端着两碗肉丝面。
<br/>　　“怎么这么早醒来？”萧逸眼中隐隐有一丝责怪之意。
<br/>　　依依昨晚其实睡得很香，但心中始终记挂着要照顾萧逸，所以今天一大早就醒来，并在萧逸醒来前准备好了早餐。
<br/>　　依依听得出来，萧逸那责怪的语气中，更多的是浓浓的关心，吐了吐舌头，不语，转过身去打了一盘水给萧逸洗涮。
<br/>　　萧逸知道依依的性格，一向乖巧可人，便没有再多说什么。简单地洗涮了一番后，吃起了早餐。
<br/>　　不得不说，依依的厨艺很不错，明明只是简单的东西，在她手上却施了魔法般，变得很是美味。
<br/>　　而且，食物虽然算不上丰富，却营养均衡。
<br/>　　“待会儿吃完早餐，把这两粒东西吃了。”萧逸边吃着粥，边把两粒丹药放到台面。
<br/>　　一颗强身丹，一颗养气丹。
<br/>　　这两种丹药对武者的有一定的作用，但不大，反倒是对普通人有着极大的功效。
<br/>　　强身丹，顾名思义就是让身体变得更加健康。
<br/>　　养气丹，补气盈虚。
<br/>　　依依多年来吃不饱穿不暖，导致身体很是虚弱。
<br/>　　而这两种丹药，将如同补品良药，为她疗养身体。
<br/>　　“这...这是丹药。”依依一下子认出了这两粒东西，而后连忙推辞道，“少爷，您是武者，更加需要丹药呢。”
<br/>　　“张嘴。”萧逸只是淡淡地说了一个词。
<br/>　　“阿？”依依愣了愣。
<br/>　　趁着她愣神，萧逸一把将丹药塞到她嘴中。
<br/>　　一会儿功夫，明显可以看到，依依原本的苍白脸色变得稍稍红润了一些，整个人看起来也更加有精神了。
<br/>　　“嗯，不错，有效果就好。”萧逸满意地点点头。
<br/>　　吃罢早餐后，萧逸开始了今天的修炼。
<br/>　　......
<br/>　　时间如过隙白驹，嗖的一声，快得飞起，让人没能反应过来它怎么能跑那么快。
<br/>　　充充半月，转瞬便过。
<br/>　　今天是萧家最重要的日子之一，三年一次的开启紫云洞，一年一度的家族比武，都将在今天举行。
<br/>　　如此重要的日子，让萧家所有族人都停下了手头工作，聚集到家族中，等待着这场盛况。
<br/>　　家族比武的意义在于考察家族子弟一年来的修炼情况，看他们有没有认真修炼。
<br/>　　比武中，若能取得较好的名次，将得到家族不菲的奖励。
<br/>　　而老一辈的家族武者们，也对此尤为重视。
<br/>　　毕竟，年轻一辈关系到家族日后的传承，也影响着萧家数十年后的兴衰。
<br/>　　.....
<br/>　　房中，萧逸从打坐修炼中醒来，眼中迸发出一股摄人的精光。
<br/>　　这半个月来，他每天都在刻苦修炼。可惜，修为并没有突破，仍旧只是凡境五重，距离凡境六重还有一小段距离。
<br/>　　他的嘴角，忽然溢出一道自信的笑容。似乎，除了自己的修为外，他还有别的底气，拥有足够的把握应付应付今天的家族比武。
<br/>　　“少爷，早餐准备好了。”忽然，一道曼妙的身影缓缓走入房中，莲步轻移，一举一动，让人赏心悦目。
<br/>　　少女那精致的面容，白哲的皮肤，让人深深迷醉，挪动不开眼睛。
<br/>　　来人是依依。
<br/>　　值得一提的是，经过这半个月每日吞服强身丹和养气丹，依依的身体不再虚弱后，竟变得漂亮了。
<br/>　　依依原本就很美，只不过以前常年受到打骂，又营养不良，这才给人一种‘丑八怪’的错觉。
<br/>　　“美，好美。”萧逸心里不禁赞叹一声，“就算是前世那些漂亮的大明星，恐怕在依依面前也会自惭形秽吧。”
<br/>　　当初萧逸刚刚穿越到这个世界时，单看依依的背影，已经觉得这是个绝世美人。
<br/>　　虽然后来看到正面，这是个面黄肌瘦的少女，但以萧逸阅女无数的经验，也是一眼看出依依是那种天生丽质难自弃的美人胚子。
<br/>　　果然，经过一段时间的调养，现在的依依真可谓国色天香。
<br/>　　若是给依依换上一套华丽的服饰，必然是一个活生生的古典美人，而且还是倾国倾城那种。
<br/>　　“少...少爷，你在看什么？”感受到萧逸那灼灼的目光，依依有些不自在，一阵脸红。
<br/>　　“没什么。”萧逸笑笑，收回了目光，道，“今天是家族比武的日子，要跟我去看看热闹吗？”
<br/>　　依依乖巧地摇了摇头，道，“不了少爷，我想在家等您。”
<br/>　　“好，随你。”萧逸耸耸肩，出门离去。
<br/>　　后面，依依认真地看着萧逸离去的背影，其实她也想去看热闹。不过，她知道萧逸是去比武的，回来后一定会很累。她想在家准备好饭菜，好让萧逸一回来就能吃到热腾腾的美味饭菜，缓解疲劳。
<br/>　　.....
<br/>　　来到家族比武场，萧逸发现，这里早已人声鼎沸，人群密密麻麻，起码有数千人。
<br/>　　作为紫云城三大家族之一，萧家真的是家大业大。
<br/>　　“看，是萧逸那个废物...”
<br/>　　“啧啧，那个废物竟然真的敢出现。”
<br/>　　几乎是萧逸出现的瞬间，不少家族子弟已经开始指指点点。目光中有鄙夷，有不屑，以及一丝可怜。
<br/>　　不错，可怜。
<br/>　　萧逸与萧若寒打赌的事，早已传遍了萧家。
<br/>　　在这些家族子弟看来，胆敢和萧家年轻一代的最强者打赌，萧逸的下场肯定会很惨的吧。
<br/>　　一些家族前辈看向萧逸的目光中，则是满脸的戏谑与失望。
<br/>　　“家主失踪前，可是家族里最为耀眼的天才，怎么就生了个这么不成器的儿子呢。”
<br/>　　“哎，当年的家主，天赋冠绝全族，智勇双全，可惜了，他唯一的血脉却是个脓包。”
<br/>　　不少族人都在摇头叹息。
<br/>　　这时，远处，五长老、七长老、八长老和一些与他们为首的执事们，纷纷露出了恶毒的目光。
<br/>　　“小废物，今日之后，必将你从萧家除名。”
<br/>　　“这个小废物，让他半月前嚣张，哼，今天有他好受的。”
<br/>　　三人低声交谈着。
<br/>　　另一边，萧逸感觉敏锐，显然感受到这数股不善的目光，当即看去。
<br/>　　五长老等人反应也很快，立刻收回了目光。
<br/>　　“哼。”萧逸冷笑一声，“一群小人，别以为我不知道你们心里想的什么，有我一天，你们就休想得逞。”
<br/>　　如果是以前的草包萧逸，自然看不清形势。
<br/>　　可现在的萧逸，一眼就看出了五长老的意图。
<br/>　　很显然，他早就觊觎萧家家主的位置，想独揽萧家大权。而自己这个名义上的少家主，则是阻挠他揽权的障碍。
<br/>　　这才有了当初萧若寒推萧逸落山崖。
<br/>　　还有之后萧逸侥幸未死，他则不知用了什么办法，拉拢了七、八长老处心积虑想剥除萧逸少家主的头衔。
<br/>　　一旦萧家没了少家主，他就能名正言顺地推举出新的少家主。
<br/>　　而整个萧家年轻一代，有资格当新少家主的，也就是萧若寒了。
<br/>　　若他的计谋得逞，萧若寒上位，三长老这个代理家主必将被慢慢架空权利。到时候，整个萧家都会被五长老名正言顺地掌控。
<br/>　　三长老萧重为人重情重义，但有些古板，论计谋，可比不上老奸巨猾的五长老。
<br/>　　当然，现在的萧逸，绝不会让他们得逞。
<br/>　　这时，比武台上传来一声大喝，“家族比武，即将开始。”
<br/>　　这是家族中的一位执事，后天八重武者，担任这次比武的裁判。
