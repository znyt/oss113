　　事实上，比起面具来说，还有其他几样东西更适合他，但如果兑换那些东西的话，用以兑换翅膀的积分就不够了。
<br/>　　一番犹豫过后，最终他还是选择兑换了翅膀。
<br/>　　之所以如此想要这双翅膀，是因为其功效，实在是太过于强大了。
<br/>　　名称：死亡之翼
<br/>　　评分：A级
<br/>　　种类：移植器官
<br/>　　介绍：原本是恶魔统领身上的翅膀，因尸体被盗，流传了出来，获得此翅膀的宿主，将觉醒飞行能力，且可以无条件借用恶魔的力量，（一周限三次，每次效果持续十五分钟，在此次数使用完之前，死亡之翼提供大量的速度加成。）
<br/>　　几乎是第一眼看到这个物品的时候，刘哔就已经决定兑换了，不仅仅是速度加成和借用能力的属性，最重要的，是可以获得飞行能力的属性。
<br/>　　纵观两个世界，他所见的所有能力者，不管修为再深，都不可以飞行，就算是先前的女鬼，最多也只能算飘罢了。
<br/>　　有传闻，在他原本的末世世界中，异能修炼到九级才可以觉醒飞行能力，但是九级异能者，那种神龙不见尾的存在，他根本没有见过。
<br/>　　也不过只是传闻罢了。
<br/>　　空对地的战斗是怎样的，如果同级的话，大概也只有碾压可以形容了吧。
<br/>　　可以说，有这样一对翅膀，他在战斗一开始就会占据很大的优势，就算是遇到实力相差悬殊的敌人，他也可以从容不迫的退走。
<br/>　　如果之前在义庄的时候他有这样一对翅膀，那鬼修老头，他有无数种方法可以击杀他。
<br/>　　也不至于搞得像后来那样狼狈了。
<br/>　　“兑换物品‘死亡之翼’成功，扣除积分九千点，宿主是否现在接入？”
<br/>　　随着系统提示音的响起，刘哔积分账户上的余额变成了0点，深吸了一口气，刘哔关闭了系统界面，下达了立即接入的指令。
<br/>　　紧接着，背后传来一阵久违的瘙痒感觉，微微有些刺痛，像是有无数虫子在自己后背上的皮肤内蠕动，想要挣脱出来一般。
<br/>　　自从变成丧尸体质以来，很长一段时间他都没有体会过疼痛是什么样的滋味了，此时反倒有种恍若隔世一般的感觉。
<br/>　　有些东西，失去了之后，才会让人懂得它的珍贵，就连疼痛，此时仿佛都变成了一件幸福的事情。
<br/>　　但没过多久，他就对心中的想法后悔不已，后背上的那股疼痛感，随着时间变的愈加剧烈了。
<br/>　　仿佛是体内每一个细胞都被撕裂了一般，这股痛感，让他双眼变的通红，额头暴起一根根青色的脉络，口中不自觉的发出一阵低吼。
<br/>　　不知道持续了多长时间，后背上的疼痛感逐渐削弱，刘哔趴在桌面上，体内提不起一丝气力。
<br/>　　一滴冷汗，自他的额头慢慢滑落，浸入他那苍白的脖颈里，此时他甚至还有些模糊，没有察觉到这些许异样。
<br/>　　丧尸，是不会出汗的。
<br/>　　或许是受到‘死亡之翼’的改造影响，刘哔的体质发生了一些改变，如果细致的区分开来，他现在已经不完全属于丧尸了。
<br/>　　这种变化给他带来的后果时好时坏，却是犹未可知了。
<br/>　　大概休息了半钟的时间，刘哔稍稍恢复了一些力气，便立即从椅子上坐了起来，将心神沉入体内。
<br/>　　虽然购买‘死亡之翼’的时候并没有使用说明，但他很快就找到了翅膀的使用方法。
<br/>　　就像是身体上与生俱来的一部分，根本不需要刘哔费力寻找，就能感应到它的存在，其上还附庸着一股阴邪的力量，并不属于他，他却可以轻易驱动。
<br/>　　想到一周内只有三次使用次数，刘哔也没有浪费，只是将翅膀催发了出来，有些生涩的低空飞到屋外。
<br/>　　这双翅膀大概有两米多长，通体都是墨黑色的羽毛，在刘哔的背后悬挂着，远远看去，仿佛一只巨大的乌鸦一般。
<br/>　　因为刚开始使用，他对于翅膀的掌控并不是十分熟练，歪歪扭扭的，仿佛随时都会在半空中掉下来一般。
<br/>　　尝试着飞行了片刻，夜晚的视线并不算好，刘哔也没在外面逗留太长时间，熟悉了基本运用之后，就落地回到了屋里。
<br/>　　将翅膀重新收回到体内，背后出现了两道狰狞的疤痕，那是翅膀破体而出所留下的，对此没有丝毫察觉，刘哔打开了商城抽奖系统。
<br/>　　一个金光闪闪的转盘从他眼前浮现，转盘中心，一个阿拉伯数字‘1’，代表着可以使用的次数。
<br/>　　“系统，开始抽奖。”
<br/>　　刘哔心中默念，对系统下达了指令，随后，转盘缓缓转动，幅度越来越大，指针在众多商品之间不断盘旋着。
<br/>　　随着指针的转动，刘哔心情也开始跌宕了起来。
<br/>　　未知的东西，总是会叫人抱有很强烈的期待感，转盘这种东西，自然也不例外。
<br/>　　虽然说，这上面的大部分东西，看起来都很普通，甚至很多东西刘哔都能叫出名字，但这依旧影响不了他心中的兴奋。
<br/>　　大概十几秒钟的功夫，转盘转动的速度逐渐缓慢了下来，随后，指针缓缓停留在了一个明黄色格子里。
<br/>　　空间一阵扭曲，一个棱角分明的长条形物体，从他眼前的空气中掉落了出来，刘哔脸色有些呆滞，下意识的伸手接住，粗糙的手感有些熟悉，但和他之前所接触的又有所不同。
<br/>　　“叮~恭喜宿主获得特殊物品‘搬砖’，相关资料已复制到商城，请宿主自行查看。”
<br/>　　冰冷的系统提示音从脑海中传来，明明不带有一丝情感，却让刘哔莫名听出了一丝嘲讽。
<br/>　　他手中握着的，是一块板砖，就是那种普通的红砖，如果非要说有什么特殊的地方，就是样子出乎意料的方正，就像是机器打磨出来的一般，没有半点瑕疵。
<br/>　　但是再方正，也摆脱不了这是一块板砖的事实，刘哔脸色有些难看，但还是怀揣着一丝希望，打开了商城面板。
<br/>　　毕竟是抽奖抽出来的东西，总不可能真是一块普通的砖头吧，刘哔心中有些犹豫，满怀期待的找出了商品介绍。
