　　第215章
<br/>　　夜来，未央宫灯影阑珊，穿着仕女服的宫女影影绰绰，小黄门来来去去，却极其安静。每个人各司其职，但不发出声音。
<br/>　　御书房内，老者歪在塌上，时不时抿一口香茶。
<br/>　　在他面前，许多人在忙碌着，他们分工有序，忙碌却井井有条。
<br/>　　这个老者不是别人，正是偌大帝国的天子，九万里江山的主人，帝国明帝陛下。
<br/>　　而书房内忙碌的，则就是贴身服侍他的太监。
<br/>　　所谓太监，并不是印象中端茶倒水，铺床叠被的仆役，若将帝国比作一家超大型公司，明帝便是这公司的董事长，而太监相当于董秘的职位。
<br/>　　明帝很喜欢太监，关键是其他人太不招人喜欢。假若在文官面前，他能这样想挖鼻子挖鼻子，想翘二郎腿翘二郎腿么，那群文官能把你祖坟骂裂了。而那些武将个个拥兵自重，一副随时准备造反的架势，实在令人喜欢不起来。
<br/>　　明帝今年已经六十三岁了，在这个时代他已不算年轻，而对于一个帝皇来说他已称得上长寿。长寿对一个老人的意思是，随时可以去死了。
<br/>　　这次武举定下一百位考官，另有两位主考官，而还有一位隐形的大BOSS，就是明帝本人。今天上交来一百份考卷，这些考卷需要细读，评级，吸取其中的养分，了解如今的帝国的思潮，这些事若让一个六十三岁的老人来独自完成，实在有些强人所难。
<br/>　　所以，明帝身边的太监便派上了用场，由他们评阅，只有遇到真有价值的，或者难以决断的文章才会交给明帝，美其名曰『御览』。
<br/>　　其中又大有文章可做，究竟那些是有价值的，那些是不能脏了明帝眼睛的，就要看执笔太监的心情了。
<br/>　　明帝打了个盹儿，醒来后想起什么，忽然道：“牛三斤的文章在哪里？”
<br/>　　“陛下，怎么想到要看他的文章？”
<br/>　　“朕听说他要拿第一，倒是想看看，他心里藏着什么东西，敢夸这海口。”
<br/>　　明帝对牛三斤是有些印象的，毕竟牛三斤考官的位置就是他亲自定的。之所以如此，是因为牛三斤的祖上曾经伺候过帝国的开国皇帝。明帝每思及此，都会想，唉，还是老祖宗会选人呐，相比之下，我身边的文官武将都是些什么东西。
<br/>　　从这个角度说，没有后台的牛三斤，他的后台就是明帝陛下。
<br/>　　『浅谈亮剑精神』
<br/>　　呃……
<br/>　　这个题目令见多识广的明帝陛下也愣了愣，这批文章的文风大概分为两种，一种是文官的引经据典，长篇大论，讲究字字有处处，句句有典故，反正就是不会说人话，这种文章也是最令太监头疼的，最后看完了，觉得文章的每句话都对，但要说整篇文章是什么意思，却谁也不知道。而武将的文章却往往颠三倒四，逻辑混乱。毕竟，他们擅长使用的兵器是刀，而不是笔。而那些能文能武的高手，却又极少。
<br/>　　相比之下，这篇文章的文风就太简单了，简单得连三岁小孩都能听懂。
<br/>　　『一个剑客在与对手狭路相逢时，无论对手有多么的强大，就算对手是天下第一的剑客，明知不敌，也要亮出自己的宝剑。即使是倒在对手的剑下，也虽败犹荣，这就是亮剑精神』
<br/>　　『事实证明，一支具有优良传统的部队，往往具有培养英雄的土壤』
<br/>　　『信仰，那曾经是多么厚重的力量，是一只军队的灵魂』
<br/>　　明帝刚开始看的时候歪着，慢慢坐直身子，到后来，他甚至站起来，在房间内不停踱步。
<br/>　　“好，好，好！”
<br/>　　明帝连道三声好，拍案击节，良久不息。
<br/>　　“这牛三斤胸里果然有山河呐，怪不得他敢夸口呢，这是自信呐。”
<br/>　　以刘阿吉为首的太监们愣住了，真的好到那种地步了。
<br/>　　太监就是太监，无法理解明帝陛下的心情。帝国一直被戎族摁在地上打，战争打不赢，也就只好在精神上寻找安慰。
<br/>　　程大雷信口胡诌，歪打正着，却也正好打中明帝陛下的心思。
<br/>　　若程大雷看到这一幕，一定会惊呼：擦，原来明帝这么好忽悠啊，看来我做个奸臣比做个山贼有前途多了。
<br/>　　“这次考核就定这个程大雷为第一。”明帝陛下一挥手：“对于人才就是要重用。”
<br/>　　“这个……”
<br/>　　刘阿吉心中一惊，明帝陛下还真是心血来潮，容易冲动啊，难道他不知道我们已经拿了别家的钱。
<br/>　　“陛下，如今戎族的和谈使节马上要到，这篇文章好虽好，但杀气太重，若是被戎族谈判的使节知道您定了这篇文章为第一，怕是会怀疑我们和谈的诚意。亮剑虽好，可是大战过后，我们需要的是休养生息呐。”
<br/>　　“这样……”明帝顿了顿，发现自己即便是一国之主，也有无可奈何的事情。
<br/>　　“那就不要定他的名次了，不过这文章还是有用的，把它抄下去，也要军中好好学习一下。”明帝道：“当然，这文章还是太粗鄙些，由你帮其润润笔。”
<br/>　　刘阿吉点点头，忽然道：“小人听闻，这牛三斤在京城的住处乃是百里胜安排的，他会不会是将军府的人？”
<br/>　　“这样么？”
<br/>　　明帝呐呐道了一声，脑海里对程大雷的一丝好感，此刻转变成恶感。
<br/>　　他不是不知道手下的派系斗争，他也并非不喜欢，他只是不喜欢一方过于强大。
<br/>　　平衡，乃是帝王之道。
<br/>　　“那个，还没有小六的消息么？”歪在塌上的明帝忽然道。
<br/>　　“鹤之队和鹿之队的人已经派出，有人说在扬州见过六王子，陛下，应该很快就有消息了。”
<br/>　　明帝目光看着飘忽的烛火，眼神中一丝微不可察的担忧闪过。他是高高在上的帝国天子，也同样是一个父亲。
<br/>　　天子和父亲两个身份融在一起，一方面，天子仇恨一切敢与他争权之人，而身为一个父亲，又不可能不喜欢他的孩子。
<br/>　　而看着那些又讨厌，又喜欢的人一天天长大起来，明帝每天什么心情普通人就很难想象了。
<br/>　　而已经六十三岁的明帝，的确已经很老了。
