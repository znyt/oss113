　　第628章
<br/>　　程大雷携了一千兵丁，随行只带着徐神机与刘发财两人，今天只是过来看看朔方城的情况。
<br/>　　骑着黑牛入城，程大雷立在城门口举目四望，一时间忍不住破口大骂。
<br/>　　“宋伯康欺我！”
<br/>　　入目都是破败的房屋，街道两旁的商铺一个开门的也没有。这倒也罢了，程大雷也不会幻想宋伯康会将一座繁华的城池送给自己。不过，朔方城破落得实在有些离谱。
<br/>　　关键是城里一点人气也无，倒也不是说城里一个人也没有，程大雷的确看到几个乞丐窝在墙角晒太阳，不时发出沉重的咳嗽声，让整座城显得死气沉沉。
<br/>　　“大当家……”徐神机发现程大雷脸上明显变得不好看。
<br/>　　程大雷回过神来：“走，继续看看。”
<br/>　　三人信马由缰，在城里转了一圈，以城池面积来说，朔方城要超过蛤蟆城许多，但城墙很多地方都坍塌了。城内也是一片破败，三人逛了一圈，来到朔方城的城主府。
<br/>　　如程大雷所想的一模一样，城主府也是一个人影也无，杯碗茶碟散落一地，可想他们离开时的匆忙。
<br/>　　程大雷翘着脚坐在大厅的帅案后，出发时的兴奋激动到此刻已烟消云散。朔方城的情况，当真糟糕得超出他想象。
<br/>　　“刘发财，你去城里走一走，逮个喘气的问一问，城里究竟是什么情况，为什么会搞成这个样子。”
<br/>　　刘发财答应一声去了，程大雷唉声叹气一番，又拽着徐神机去了城主府的账房。
<br/>　　其实查账也无什么用，现在的朔方城完全是一座空城，宋伯康把粮草、武器都搬走了，完全是抱着不留寸草与敌耕的心思。所谓的账本也是一本空账，不过多多少少还是能从中看出些什么。
<br/>　　程大雷翻出了朔方城的人口薄，查到原先朔方有人口五万户，算起来该有二三十万人，可程大雷奇怪的是，这些人都去了那里。
<br/>　　城可破，墙可以塌，关键是人。没有人就什么都没有。
<br/>　　程大雷打算让和珅过来看看，他并没有治理内政的本事，从这些账薄上也看不什么，至于徐神机就更不用提了。
<br/>　　过不多时，刘发财从外面回来，一路找到账房。
<br/>　　“大当家，我找到一个老人问了问，事情都打听清楚了。”
<br/>　　“喔，具体什么情况。”程大雷问道。
<br/>　　“原先此地的城主名叫孙博山，是宋伯康派过来的人。他这个人不是很地道，做城主这几年把城里的百姓搜刮一空，城里平民死走逃亡许多。宋伯康这次把朔方城让给我们，之前将城里能运走的东西都运走了。颁布的命令，就是寸铁不留。”
<br/>　　“这我已经知道了”程大雷看着空空荡荡的城主府，还有什么想不到的：“人呢，我问的是人，城里的人都哪里去了？”
<br/>　　“之前被孙博山折腾一通，城里的百姓走了不少。这次大家听说是大当家接管朔方城，城里的百姓闻风而动，但凡能动弹的走了。留下的都是一些老弱妇孺，以及不怕死的。”
<br/>　　程大雷一阵无语，虽然刘发财没有挑明，但程大雷也明白他话中的意思。自己的名声毕竟不怎么好，原先的百姓知道是自己这吃人的魔王来了，怎么可能还留在这里。
<br/>　　“大当家，还有一件事，城里还有一些老弱，大部分都是活不下去的人，家里早就断粮了。如果咱们不管他们，他们便都得饿死。这件事…”
<br/>　　程大雷拍拍脑袋，觉得有些头疼。果然呐，天上掉下来的馅饼，不是那么容易吃的。冷不防就会被馅饼中藏着的石头咯掉牙。
<br/>　　“辛苦你再跑一趟，回蛤蟆城将和珅请过来，看怎么整治这座城。”程大雷道。
<br/>　　程大雷窝在桌子后，思量着接下来该做的事情。每想到一件，便随手在纸上写下来。
<br/>　　首先，城墙要修，城里坍塌的房屋也要修。这倒还是次要的，关键还是人，只要有了人，城里的农业商业手工业才能发展起来，一座将死的城池才能重新活过来。
<br/>　　可这每一件事，都是极大的消耗。就算有流民迁入朔方城，要自给自足至少等到秋天，而在这些日子他们的吃穿住行，就完全需要程大雷负责。
<br/>　　这些年程大雷辛辛苦苦，算是攒下些家底，但朔方城一件事，很有可能把程大雷攒下的家底消耗一空。
<br/>　　现在的程大雷还意识不到这是宋伯康的计策，但他已经明白这件事情的难处。可让他将白白得来的城池放弃，他暂时也舍不下。
<br/>　　或许，宋伯康赌的就是程大雷的贪心。
<br/>　　当然，朔方城也并非一点优势没有。城池依河而建，河滩有良田无数，地理位置也比较重要。而蛤蟆城目前的确到了发展的瓶颈期，想要扩张势力，程大雷需要开新的基地。
<br/>　　今夜，程大雷和徐神机就歇在朔方城，程大雷守着一盏油灯，口中长吁短叹。
<br/>　　“大当家，如果真的这么为难，不若舍了朔方城，咱们明天就回蛤蟆城。”
<br/>　　“白白舍了岂不是可惜。况且，和珅说的话也未必没有道理。想要自保也需要有自保的实力。凭蛤蟆城那点家底，怕是撑不过这个乱世呐。”面对徐神机，程大雷还是能说几句心里话的。
<br/>　　“军师，我问你一句话？”程大雷忽然道：“你觉得，咱们这帮人日后能取了天下么，各个封疆裂土，成一方诸侯？”
<br/>　　野心这种东西，程大雷也并非一点没有，他只不过是还没下定决心。
<br/>　　徐神机皱眉想了想，道：“依我看，取天下的事情谁也说不好。说不得那天好事就砸在咱们兄弟头上了。话说回来，如果大当家真的这么为难，干脆舍了这份家业，还是咱们兄弟几个占山为王，也不是逍遥自在一辈子么。”
<br/>　　程大雷心中一暖，垂下头继续翻着面前的账薄。真如果舍掉蛤蟆城，他、徐神机几个浪迹江湖，说实话要比如今逍遥许多。可家大业大，程大雷也不可能像从前那样自在。
<br/>　　总是要想个法子，有实力撑过这个乱世。
<br/>(本章完)
