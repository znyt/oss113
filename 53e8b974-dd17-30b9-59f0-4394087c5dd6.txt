　　第29章
<br/>　　“当时你们是没看见呐，他大喝一声，那大石头就嘭地一声爆开。”
<br/>　　在飞虎寨，高飞豹绘声绘色描述着那天的事情经过。
<br/>　　“说实话，我们都被吓着了，这个程大雷可真的会五雷法啊，太厉害，太厉害了。”
<br/>　　高飞虎脸上的表情凝重起来，目光望着身边坐着的一中年男人。
<br/>　　“军师，依你看这是怎么会事？”
<br/>　　中年男人呵呵一笑，手指着桌上搁着的一块石头，石头上有淡淡的黑迹。
<br/>　　“这块石头是我从峡谷中捡来的，上面还有爆炸留下的痕迹，你们看看，能看出什么嘛？”
<br/>　　高飞虎两兄弟凑到石头前，大眼瞪小眼，同时摇摇头。
<br/>　　“这是火药爆炸形成的痕迹，哪里有什么五雷法，你们都被唬住了。”军师笑道。
<br/>　　“火药？”高飞虎面露困惑。
<br/>　　“就和爆竹中的药粉差不多，不过威力更大，就算在外面也很罕见，何况青牛山这么偏僻，你们不知道也不奇怪。”
<br/>　　高飞虎和高飞豹面面相觑，想到自己被程大雷用这种把戏唬住，二人都有些恼羞成怒。
<br/>　　“混蛋，我现在就带人打下蛤蟆寨，竟然敢骗我。”
<br/>　　“二当家别急，别急。”军师笑着拦住高飞豹：“那蛤蟆寨就在这里，也跑不了，何必着急呢。”
<br/>　　这军师名叫于虬然，三年前加入飞虎寨，读过书，很有计谋，在飞虎寨，高飞虎和高飞豹对他可以说是言听计从。这段时间于虬然不在飞虎寨，因此高飞虎兄弟许多事都没有主张。
<br/>　　“可这口气憋在心里真特么难受。”高飞豹气鼓鼓道。
<br/>　　“早晚的事情，早晚的事情。”于虬然道：“眼下，正有一笔大买卖要我们去做，蛤蟆寨的事先放一放。”
<br/>　　“什么大买卖？”一听这话，高飞豹来了精神。
<br/>　　……
<br/>　　蛤蟆寨。
<br/>　　程大雷看着刚收到的两条系统提示，有些无语。奖励一亩成熟辣椒……现在他已经摸清楚系统套路，这一亩辣椒说不定在那个山坡长着，可是这奖励委实有些水，水到程大雷把它找出来的兴趣都没有。
<br/>　　倒是触发的打劫任务有些搞头，只是不知道完成后会有什么奖励。
<br/>　　想着，程大雷打开系统界面，查看自己目前待完成的任务。
<br/>　　任务一：用非暴力手段打动苏樱，奖励未知。
<br/>　　任务二：招募山寨人口达到100，奖励抽奖次数1 。
<br/>　　任务三：完成10次打劫，奖励未知。
<br/>　　现在程大雷已经明白，想要尽快将山寨发展起来，还要依靠系统，依靠系统发布的任务。不停完成任务，不停获得奖励。
<br/>　　目前自己拥有的三个任务，招募人口暂且没有办法，打劫任务可以交给秦蛮和林少羽完成，那么，也就只剩下最后一个任务——用非暴力手段打动苏樱，完成抢夺压寨夫人的第二阶段。
<br/>　　这还是程大雷接到的第一个多环任务，如果全部完成，想也知道奖励一定很丰富。
<br/>　　当然，这并不是程大雷想要首先完成这个任务的最主要原因，最主要原因是——苏樱长得好看呐。
<br/>　　和窝在草丛中，忍受炎热，潮湿和蚊蝇叮咬，当然是面对一个美女更美妙些。
<br/>　　这个任务看似艰难，其实不然，在程大雷看来，这就是一个刷好感度的任务，就像他玩过的无数游戏一样，采摘某个山谷中的花朵给NPC，当你送出一朵时，可以获得一点好感度，当你送出一百朵时，就可以获得一百点好感度，只要达到一定数值，NPC就会对你说：年轻的少侠喔，我这里有一本秘籍要交给你，世界和平的任务就交给你了……
<br/>　　当然，山贼之心并不会给自己具体数值，不过只要不停刷下去，总能触发隐藏任务的。
<br/>　　对于打动苏樱的心，程大雷还是有自信的。他可不是那种一根肠子到底的理科宅，他是一个——文科宅。好吧，这似乎并不能带来优越感，不过至少自己也看过数不清的电影，知道许多催人泪下的爱情桥段。
<br/>　　程大雷将这当做一场战役，他把秦蛮和林少羽派到山下打劫，自己做足充分准备，开始完成这个任务。
<br/>　　其实，这些天，程大雷也不在限制苏樱的自由，她可以随意在山寨中行走。程大雷也不担心她会逃跑，青牛山山贼众多，如果被别的山寨掳走，程大雷不觉得有任何一个山贼比自己绅士。
<br/>　　而这个时候，程大雷的刷好感度任务已经开始执行。
<br/>　　比如突然跳到苏樱樱面前，大声道：“苏樱樱，我宣你！”
<br/>　　苏樱樱视若无睹，从程大雷身边经过。程大雷有些无语，桥段不该是这样的，当自己大声喊出这句话的时候，她不是该问：“谁，谁宣苏樱樱。”
<br/>　　然后自己就可以告诉她：“程大雷宣苏樱樱。”
<br/>　　计划A失败，不怕，程大雷还有无数计划，无数桥段。
<br/>　　“喂，我养你啊。”当苏樱在程大雷面前走过时，突然喊住他。
<br/>　　“咦，这个妹妹我是在哪里见过的。”突然跳到苏樱面前，假装面带困惑。
<br/>　　“世界上好人不少，不过你是最重要的一个。你要是愿意，我就永远爱你，你要不愿意，我就永远相思。”手捧一朵花束，满目深情的冲苏樱说。
<br/>　　“夜阑卧听风吹雨，铁马是你，冰河也是你。”偷偷在苏樱的房间里塞纸条。
<br/>　　“醒来，觉得甚是爱你。”故意在大清早没睡醒的时候敲开苏樱房门。
<br/>　　程大雷就像失控的马车，在一条刷好感度的路上狂奔。可是，苏樱没有任何反应，唯一的表情就是睁着无辜的大眼睛，欣赏着程大雷的表演。一开始，程大雷觉得苏樱就算是冰封的湖面，自己只要疯狂的砸进石头，总会将冰层砸破。
<br/>　　然而他现在明白，苏樱根本就是一道深渊，不管你丢进去多少石头，也不会有任何回应。
<br/>　　直到有一天，程大雷满怀柔情的望着苏樱：“曾经有一段真挚的感情摆在我面前，我没有去珍惜……”
<br/>　　苏樱眨着她那双雪亮的眼睛，认真询问道：“你最近……是病了吗？”
