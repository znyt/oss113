　　闭关室内。
<br/>　　萧逸仅仅修炼了三天，便已出关。
<br/>　　不过，除了刚开始之时修炼了帝岳不灭体，且炼至小成后，之后再无修炼。
<br/>　　之后的时间，他都在修炼他的星幻剑道。
<br/>　　星幻剑道，越是往上修炼，其难度便越大。
<br/>　　这不仅来自于星幻剑道本身的强大，还来自于其层次的提升。
<br/>　　掌控八万星光，其武道层次，已达圣皇境八重。
<br/>　　也就是说，他现在是位于圣王境六重的武道修为，去参悟圣皇境八重以上的武道规则力量。
<br/>　　自然的，这难度变得极大。
<br/>　　整整三天闭关，他也不过是多掌控了数百道星光之力。
<br/>　　上次那三月闭关，之所以掌控星光的数量如此多，主要还是来自于星幻瓶的相助。
<br/>　　现今正常修炼，而且难度如此大，他便暂时选择停下修炼。
<br/>　　另外，他也没必要闭关了，便也出关了。
<br/>　　帝岳不灭体的修炼，他早已停下。
<br/>　　里头中间部分的内容，他还是需要斟酌几分，之后再考虑是否修炼下去。
<br/>　　那20块修罗太荒石，他也未动过。
<br/>　　一来，不需要这些精纯的土属性力量。
<br/>　　二来，是否要动这份乾坤戒内的大礼，他还不确定。
<br/>　　先不说里头的大量土属性重宝、天材地宝等等。
<br/>　　单说这20块珍贵无比的修罗太荒石。
<br/>　　一块修罗太荒石，就抵得过一条灵脉的力量。
<br/>　　但，灵脉，是谁都能用。
<br/>　　而修罗太荒石，则是只适合修习土之一道、山石一道的武者。
<br/>　　看似修罗太荒石局限性比较大。
<br/>　　但实则，对于修习土属性一道的武者来说，这乃是至宝，其珍贵程度，远远高于一条灵脉。
<br/>　　效果也远高于灵脉。
<br/>　　特别土属性一道，最是坚如磐石。
<br/>　　故修习此道者，不乏体修武者。
<br/>　　对于体修武者来说，一块修罗太荒石的价值，远超灵脉。
<br/>　　另外，他曾在修罗殿的兑换手册上，看到过修罗太荒石的信息。
<br/>　　即便是总殿的那块宝地，也不过是一年产5块左右。
<br/>　　若是用任务点和功绩去兑换，更是一个极其庞大的数字。
<br/>　　现今，总殿主足足给了他20块。
<br/>　　算上其他的土属性重宝、天材地宝等等。
<br/>　　这个乾坤戒，简直就是一份大礼。
<br/>　　所谓无功不受禄，无端端的，萧逸可不敢随便要这份大礼。
<br/>　　走出闭关室。
<br/>　　萧逸霎时看到闭关室外一位执事迎面走来。
<br/>　　“萧逸副殿主，这么快便出关了？”执事笑脸相迎道。
<br/>　　“呵呵，真不愧是天纵之资，这么快就能参悟功法。”
<br/>　　“额，倒不是。”萧逸摇了摇头。
<br/>　　“只是提前出关罢了。”
<br/>　　“可是总殿主要召我？”
<br/>　　萧逸问道。
<br/>　　他记得，闭关前，总殿主说过，让他出关后去他那一趟。
<br/>　　“是。”执事点点头，“总殿主吩咐我在此等候，萧逸副殿主一出关，便召你前去。”
<br/>　　“嗯，带路吧。”萧逸点点头。
<br/>　　……
<br/>　　修罗殿内堂。
<br/>　　属于总殿主处理事务的书房内。
<br/>　　那执事推门而入，萧逸跟随而进。
<br/>　　果然，里头总殿主正批阅着卷宗。
<br/>　　“嗯？出关了？”总殿主对于萧逸的到来，显然有些惊讶。
<br/>　　毕竟，萧逸才仅仅闭关了三天。
<br/>　　“仅仅三天时间，便完全参悟了吗…”
<br/>　　“不对。”总殿主眉头一皱，“你的肉体力量并未提升多少。”
<br/>　　总殿主，乃是当世体修第一人，自然一眼便看穿了萧逸的肉体力量层次。
<br/>　　执事对着总殿主行了一礼，躬身离去。
<br/>　　房内，只剩总殿主与萧逸二人。
<br/>　　“可是帝岳不灭体上，有什么不懂的地方？”总殿主率先问道。
<br/>　　他那威严的面庞，即便只是询问着，仍旧给人一种严肃无比的感觉。
<br/>　　“不是。”萧逸摇了摇头。
<br/>　　“不是？”总殿主微微皱起了眉头。
<br/>　　他自然知道萧逸选的是帝岳不灭体。
<br/>　　而现今萧逸如此快出关，在他看来，也只有遇到修炼问题这一个可能。
<br/>　　毕竟，闭关室内早已准备好足够的修炼之物。
<br/>　　若是能修炼，根本不会这么快出关。
<br/>　　他没想到的是，萧逸之所以如此快出关，便是因为那大量的修炼之物。
<br/>　　萧逸缓缓走到批阅卷宗的桌案前，行了一礼。
<br/>　　“非是小子修不了那帝岳不灭体，只是…”
<br/>　　萧逸迟疑了一下，随后直言之。
<br/>　　“不敢收？”总殿主先是一愣，随后威严的面庞上，闪过一丝轻笑。
<br/>　　“我以为是什么。”
<br/>　　总殿主直视萧逸，轻笑道，“若不为你准备好足够的修炼之物，让你在短时间内提升修为。”
<br/>　　“你如何代表我修罗殿去参悟古帝之墓的开启？”
<br/>　　“什么？”萧逸脸色一惊。
<br/>　　“总殿主的意思是…”
<br/>　　“不错。”总殿主点了点头，“修罗殿的名额，归你。”
<br/>　　“可是…”萧逸想说，他已经有名额了，而且是风刹殿的名额。
<br/>　　总殿主摆摆手，打断道，“你本就有风刹殿的名额，也会代表风刹殿前往古帝之墓，此事我知晓。”
<br/>　　“风刹殿总殿主，已与我说明。”
<br/>　　“不过，这并不影响你同时代表修罗殿前往古帝之墓。”
<br/>　　萧逸皱了皱眉，“恕小子直言，我不是太明白。”
<br/>　　“明白些什么？为何名额给你？”总殿主问道。
<br/>　　“我修罗殿，此番并无年轻天骄前往，若不给你，也只是浪费掉，便给你了。”
<br/>　　萧逸愣了愣。
<br/>　　总殿主的回答，与风刹殿总殿主的回答，一模一样。
<br/>　　只是，直觉告诉萧逸，绝不单单是如此。
<br/>　　果然。
<br/>　　总殿主继续道，“你代表风刹殿前往古帝之墓，顶多算是一份历练。”
<br/>　　“所以，你即便再代表修罗殿前往，也并无影响。”
<br/>　　“不过，老夫有一事要拜托你。”
<br/>　　“这…”萧逸眉头一皱。
<br/>　　不仅是讶异于总殿主会拜托他。
<br/>　　更是皱眉于总殿主可能会说些什么。
<br/>　　这个层次的人物，所拜托的事，岂可能会是等闲之事。
<br/>　　“总殿主高看小子了。”萧逸并未给总殿主说话的机会，抢先道。
<br/>　　“小子不过是个名不经传的普通武者，可当不得总殿主的拜托。”
<br/>　　“若无其它事，小子便告辞了。”
<br/>　　若无必要，萧逸并不想卷入一些麻烦之内，特别是这个层次的麻烦。
<br/>　　他已经够时间紧迫的了，没兴趣在别的地方浪费时间。
