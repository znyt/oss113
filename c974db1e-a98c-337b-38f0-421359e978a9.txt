　　那片红色内，如同隐藏着滔天的血海，给每一个见过的人都会留下极为深刻的印象。
<br/>　　此事过后的很长一段时间，秋生都不敢独自一个人待在黑暗里。
<br/>　　见到两人脱离了生命危险，九叔松了口气，嘱咐一声，让两人退到一旁，和自己的师弟两人联手，一起对抗剩下的那些僵尸。
<br/>　　看到九叔粗鲁的动作，四目道人满脸心疼，这些僵尸可都是他的顾客，万一磕碰了边边角角，到时候这些僵尸的家属追究起来也是麻烦。
<br/>　　片刻的功夫，堂内的战斗逐渐停息下来，九叔轻轻舒了一口气，将跑的到处都是的僵尸扛回了原处。
<br/>　　“走，我们过去帮忙。”刘哔眼中的红色已经淡了下去，看了一眼愣在原地，不知道在想什么的秋生，招呼一声，主动走了上去。
<br/>　　他心中觉得有些奇怪，不知道为什么，总觉得秋生看自己的眼神不太对劲，这个家伙，不会有一些特殊的癖好吧。
<br/>　　没来由的涌上一股恶寒，刘哔加快脚步，和秋生拉开了一些距离。
<br/>　　气氛有些沉重，九叔阴沉着脸，将桌子上打翻的灯座重新立起，从兜里掏出一盒火柴，划了好几根都没有引起火来。
<br/>　　“兔崽子！”嘴里嘀咕一声，九叔狠狠瞪了正在贴黄纸的文才一眼，幸亏他之前和师弟在院子里说话，否则的话，也来不这么及时。
<br/>　　到时候，说不准会发生什么样的事情。
<br/>　　四目道长看了看自己正在生气的师兄，三两步走上前去，轻轻拍了拍九叔的肩膀，拿出一盒崭新的火柴，重新给油灯续上火头。
<br/>　　“唉，这两个兔崽子，开什么玩笑啊，拿我的顾客来玩啊？”一切准备妥当之后，四目道长站在门口，手中的铃钟，随着他的动作发出一连串清脆的声响。
<br/>　　文才满脸苦色，脸上的褶子都皱成一团，伸手指向一旁的秋生，正欲开口辩解，却被四目道长接下来的话语打断。
<br/>　　“不要说了，我走了。”四目道长伸手，阻止了文才的话语，反正事情已经解决，他也不想继续听这两个人在这里啰嗦。
<br/>　　秋生拉了拉文才的衣角，有些恼怒，但是想到师父还在这里，便也不在争论，只是用眼睛瞪了文才一眼，气恼的站在原地。
<br/>　　“多住两天嘛。”九叔伸手，脸上的表情不像客套，他和自己的师弟经常见不到面，这次一别，再相见的时候就不知道是何时了。
<br/>　　稍稍犹豫了一下，四目道人摆手，没有太多言语，和九叔告别一声，便摇着手上的铃钟走了出去。
<br/>　　随着清脆的铃声响起，额头上贴着黄符的僵尸，像是遵循着某种节奏一般，整整齐齐的蹦跳出去，刘哔在一旁静静观看着，觉得这件事情有些稀奇。
<br/>　　在他原本的世界中，末世已经爆发了十年，什么样的奇异事件他都见过，但眼前的一幕，又重新激发了他的认知。
<br/>　　“这么晚了还不回去做什么？不然你姑妈又要过来要人了，回去！”打开义庄的大门，九叔走进屋内，双手叉腰，语气严肃的对秋生说道。
<br/>　　乖巧的应了一声，秋生看了一眼刘哔，欲言又止，转头跑出了大厅。
<br/>　　看着秋生慌慌张张的身影，九叔叹了口气，和刘哔一前一后站在房间的门口，门外，一直身材矮小的僵尸被门槛卡住，文才正扶着它出门。
<br/>　　“其实吧，文才的品行倒是挺不错的，但这个兔崽子，脑子实在是不太灵光。”背对着刘哔，九叔喃喃的说道：“可是，秋生又太爱惹祸，再加上他姑妈一直反对他学道术，所以到现在，九叔的衣钵都传不下去啊。”
<br/>　　刘哔微低着头，口中不时传来一阵瘙痒的感觉，让他忍不住用舌头轻轻舔舐着自己的獠牙，对于九叔的这番话，他却也没有听进去多少。
<br/>　　此刻，他所有的注意力都放在了系统任务上面，之前见到九叔和四目道长的时候，系统有颁布过一个任务，但当时的情况有些匆忙，他还没抽出时间查看。
<br/>　　此刻，所有事情都解决完毕了，他也终于能沉下心来查看自己的任务面板了。
<br/>　　名称：学艺
<br/>　　内容：得到九叔的衣钵传承。
<br/>　　等级：D+
<br/>　　时限：无。
<br/>　　失败惩罚：无。
<br/>　　备注：所有任务分为从A至F六个等级，每个等级有+-三个小段，任务等级越高，奖励会越丰厚。
<br/>　　任务面板上的内容一目了然，但刘哔心中却没有多少欢喜，先不说八字还没一撇的任务奖励，光是任务的内容想要完CD十分困难。
<br/>　　要知道，九叔的身边的两个徒弟，跟他的时间要比自己长出许多，一直到现在都还没有得到九叔的传承，更不要说刚刚拜入九叔门下的他了。
<br/>　　再说想要学习道术，需要一个循环渐进的过程，就算九叔真的肯教自己，那学习的过程也是相当漫长的。
<br/>　　而九叔肯定不会将自身会的所有道术，一股脑的全都教给他，按照常理来说，他总得一个个进行学习，这更让任务增加了一些难度。
<br/>　　不过好在，系统颁布的这个任务没有时限性，就算真的失败了，也不会有什么惩罚，如果真的没有机会完成，大不了就是不做了，对他也没有任何损失。
<br/>　　仔细琢磨了一遍，刘哔退出系统任务面板，而此时文才和九叔，早就离开了院子，刘哔舔了舔獠牙，转身向自己的房间走去。
<br/>　　先前出来的时候并没有多加注意，此刻闲暇的时候，倒意外发现这个义庄还是蛮大的，庄内房屋交错，由一条长廊连接到一起，壁画上的门神，满脸狰狞，更是增添了一丝诡异的气氛。
<br/>　　长廊外的院子里，杂乱的种植着一些草木，零星盛开着几朵长相怪异的鲜花。
<br/>　　这种鲜花娇艳无比，通体红色，或许是因为和人脸有七分相似的花盘，明明是很美丽的景色却给人带来一种诡异的气氛。
<br/>　　他之前并没听说过类似的品种，或许是专属于这个世界的特殊产物，只是，刘哔莫名感觉脊背有些发凉，忙不迭的疾走了两步，回到自己的房间。
